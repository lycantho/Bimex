## 🎯 Guías rápidas

### Para contribuyentes
1. Lee el [FAQ](faq-es.md)
2. Sigue la [Guía del contribuidor](guia-contribuidor.md)
3. Conecta tu wallet y contribuye

### Para creadores de proyectos
1. Lee el [FAQ](faq-es.md)
2. Sigue la [Guía del creador de proyecto](guia-creador-proyecto.md)
3. Prepara tus documentos y lanza tu proyecto

### Para desarrolladores
1. Lee la [Guía técnica de integración](guia-tecnica-integracion.md)
2. Revisa [DOCUMENTACION.txt](../DOCUMENTACION.txt)
3. Explora el código en [bimex/contracts/bimex/src/](../bimex/contracts/bimex/src/)

### Para el equipo Bimex (proyecto piloto)
1. Lee el [RESUMEN EJECUTIVO](RESUMEN-PROYECTO-PILOTO.md)
2. Abre el [Checklist ejecutivo](checklist-ejecutivo-piloto.md)
3. Sigue el proceso paso a paso
